# EcoSystem - Замкнутая экосистема в Unreal Engine 5

## Описание проекта
Проект представляет собой симуляцию замкнутой экосистемы с взаимодействующими видами животных. Реализовано на C++ с использованием Unreal Engine 5.

```
│   ├── EcoSystem.Target.cs          # Таргет сборки игры
│   ├── EcoSystemEditor.Target.cs    # Таргет сборки редактора

│       ├── EcoSystem.Build.cs       # Конфигурация модуля

│       └── Classes/

│           └── ECOEnvironmentManager.h/.cpp # Менеджер среды
└── EcoSystem.uplugin            # Дескриптор плагина/проекта
```

## Основные компоненты

### 1. AECOCreature
Базовый класс для всех живых существ в экосистеме.

**Характеристики:**
- Типы существ: Травоядные (Herbivore), Хищники (Carnivore), Всеядные (Omnivore)
- Параметры: здоровье, голод, энергия, скорость, радиус восприятия
- Поведение: поиск пищи, размножение, блуждание

**Основные функции:**
- `SearchForFood()` - Поиск ближайшей цели (еды)
- `Eat(float NutritionValue)` - Потребление пищи
- `Reproduce()` - Размножение при достаточном уровне энергии
- `Wander()` - Случайное перемещение по среде
- `UpdateVitals(float DeltaTime)` - Обновление жизненных показателей
- `Die()` - Смерть существа

### 2. AECOEnvironmentManager
Управляющий класс экосистемой.

**Функционал:**
- Инициализация экосистемы с заданными параметрами
- Спавн различных видов существ
- Мониторинг популяции
- Контроль перенаселения (culling)
- Статистика популяции в реальном времени

**Настройки:**
- Размер карты (MapWidth, MapHeight)
- Список видов с параметрами спавна
- Режим отладки

## Механики экосистемы

### Пищевая цепочка
- **Травоядные**: Ищут растения (в базовой реализации - любые объекты)
- **Хищники**: Охотятся на травоядных
- **Всеядные**: Могут потреблять как растения, так и других существ

### Жизненный цикл
1. **Рождение**: Спавн через EnvironmentManager или размножение
2. **Рост**: Накопление энергии через питание
3. **Размножение**: При достижении порога энергии создается потомок
4. **Старение**: Постоянный рост уровня голода
5. **Смерть**: При нулевом здоровье или перенаселении

### Баланс экосистемы
- Автоматический контроль численности популяции
- Естественный отбор через механику голода и хищничества
- Динамическое изменение популяций разных видов

## Сборка проекта

### Требования
- Unreal Engine 5.3 или новее
- Visual Studio 2019/2022 (Windows) или Xcode (Mac)
- C++ Build Tools

### Шаги по сборке

1. Скопируйте папку `EcoSystem` в директорию проектов Unreal Engine
2. Откройте файл `.uproject` (если используется как проект) или подключите как плагин
3. Перегенерируйте файлы проекта:
   ```bash
   # Windows
   Right-click on .uproject -> Generate Visual Studio project files
   
   # Linux/Mac
   ./GenerateProjectFiles.sh
   ```
4. Откройте решение в Visual Studio и соберите проект
5. Запустите редактор Unreal Engine с проектом

## Использование в редакторе

1. Добавьте на уровень актёр `ECOEnvironmentManager`
2. Настройте параметры среды в деталях:
   - Размер карты
   - Список видов (SpeciesList)
3. Для каждого вида укажите:
   - Название
   - Класс существа (Blueprint или C++)
   - Начальное количество
   - Тип питания
4. Запустите симуляцию (Play in Editor)

## Расширение функционала

### Добавление нового вида
1. Создайте Blueprint-класс, наследующий от `ECOCreature`
2. Настройте уникальные параметры (скорость, здоровье и т.д.)
3. Добавьте вид в `SpeciesList` менеджера среды

### Новая механика поведения
1. Расширьте класс `AECOCreature`
2. Переопределите методы `SearchForFood`, `Wander` и другие
3. Добавьте новые состояния и логику принятия решений

### Визуализация
- Добавьте меш и анимации для существ
- Настройте материалы и эффекты
- Используйте систему частиц для следов, крови и т.д.
